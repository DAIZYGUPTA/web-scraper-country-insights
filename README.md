# web-scraper-country-insights

Web Scraper for Country Data 
A Python-based web scraper that extracts country-level data (population, capital, and area) using BeautifulSoup, performs data cleaning, and exports structured datasets for analysis.

# Features
Extracts country-level data using BeautifulSoup
Uses CSS selectors for robust parsing
Handles missing values safely
Converts raw text into structured numerical data

# Exports:
Full dataset (CSV)
Top 10 most populated countries
Modular and reusable code design

# Tech Stack
Python
Requests
BeautifulSoup
Pandas
